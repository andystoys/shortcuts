Typing Symbols

  • Bring up emojis/symbols: ⌘ ⌃ Space
  • ⌘: Command: place of
  • ⌃: Control: arrowhead
  • ⌥: Option: option
  • ⇧: Shift: upwards white arrow
  • ⇪: Caps Lock: upwards white arrow from bar

    Finder

  • Bring up Searching "This Mac" in Finder: Cmd Opt Space
  • Desktop: Cmd Shift D
  • Home: Cmd Shift H
  • Airdrop: Cmd Shift R
  • Recents: Cmd Shift F
  • Documents: Cmd Shift O
  • Downloads: Cmd Opt L

    • Viewing

  • Toggle sidebar: Cmd Opt S
  • Icons: Cmd 1
  • Lists: Cmd 2
  • Columns: Cmd 3
  • Gallery: Cmd 4

  • Move Item: Cmd C then Cmd Opt P
